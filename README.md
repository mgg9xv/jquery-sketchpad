## Setting up the project

1. Clone the git repository using a bash terminal

    ```bash
    git clone https://github.com/mggwxyz/pixelpad.git
    ```
    
1. Navigate into the project directory

    ```bash
    cd pixelpad
    ```
1. Start the application
    ```bash
    npm start
    ```
